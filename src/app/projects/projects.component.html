Mandelbrot

A multithreaded Java program that allows you to save an image of the Mandelbrot set. Choose coordinates, zoom level, and thread count for rendering. The code and documentation is available on GitHub.

WalkSC

Senior design capstone project with Clemson's CAFLS and Cooperative Extension to develop a mobile app for the WalkSC program. Built in React Native with a Python API and SQL database on AWS.

Fractals.tech

Hackathon team submission at Clemson's CUHackit 2022 that won best use of a .tech domain. Create different types of fractals, modify parameters, and download images. Built with HTML/CSS, Python for math, and FastAPI. Check it out at fractals.iapetus.me.

Portfolio Website

This website! Originally written in HTML/CSS, then rebuilt with React, and now Angular. Hosted on AWS using S3, CloudFront, Route 53, and CodePipeline for CI/CD. Source code on GitHub.

Conway's Game of Life

A Java program based on John Conway's Game of Life. Choose from preset board sizes and setups including oscillators, still lifes, gliders, spaceships, or random boards. Control simulation speed, pause, step through rounds, and click to toggle cells. Source on GitHub.
